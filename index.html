AI Powered Public Transport Safety Platform

SafeTN is a smart digital safety monitoring platform focused on Tamil Nadu Public Transport, with primary focus on Coimbatore city. It integrates AI monitoring, live location tracking, safety analytics, emergency reporting, and transport database management.

View Project Proposal

Core Features

📍 Live Bus & Train Tracking

Real-time GPS tracking of buses and trains across Tamil Nadu.

🚨 Emergency Reporting

One-click safety alert system with location sharing.

🎥 AI Camera Monitoring

Future AI integration for behavior detection & safety monitoring.

📊 Data Analytics Dashboard

Safety score analysis and risk-based transport insights.

📝 Complaint Management

Digital complaint filing with tracking and resolution system.

👮 Admin Control Panel

Transport authority dashboard for monitoring and decision-making.

Technology Stack (Planned)

Frontend: HTML, CSS, JavaScript / React

Backend: Node.js / Python Flask

Database: MySQL / MongoDB

AI/ML: Deep Learning, Computer Vision, Data Analytics

Cloud Hosting: AWS / Render / Firebase
